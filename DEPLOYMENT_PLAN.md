## 🔐 업비트 API 준비사항

### 1. 계정 준비
- [ ] 업비트 계정 가입
- [ ] 본인인증 완료
- [ ] 2FA (OTP) 설정

### 2. API 키 발급
```
1. 업비트 웹사이트 로그인
2. 마이페이지 → Open API 관리
3. "새로운 API 키 발급" 클릭
4. 권한 설정:
   ✅ 자산 조회 (필수)
   ✅ 주문 조회 (필수)
   ✅ 주문하기 (매매용)
   ❌ 출금하기 (보안상 비활성화)
5. IP 주소 등록 (보안 강화)
6. 키 생성 → Access Key, Secret Key 저장
```

### 3. 보안 설정
```bash
# .env 파일 생성
UPBIT_ACCESS_KEY=your_access_key_here
UPBIT_SECRET_KEY=your_secret_key_here
TRADE_MODE=paper  # paper 또는 real
```

## 🚀 실행 방법

### 1. 모의 투자 모드
```bash
# 설정
echo "TRADE_MODE=paper" > .env

# 실행
python main.py

# 대시보드 시작
python dashboard.py
# 브라우저에서 http://localhost:5000 접속
```

### 2. 실전 투자 모드
```bash
# 설정
echo "TRADE_MODE=real" > .env
echo "UPBIT_ACCESS_KEY=your_key" >> .env
echo "UPBIT_SECRET_KEY=your_secret" >> .env

# 실행 (주의!)
python main.py
```

### 3. 자동 실행 (5분마다)
```bash
# crontab 설정
crontab -e

# 추가
*/5 * * * * cd /home/user/webapp && python main.py >> logs/cron.log 2>&1
```
